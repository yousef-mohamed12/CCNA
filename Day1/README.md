## Day 01 - Packet Tracer Introduction

### What this lab covers
First Packet Tracer lab from Jeremy's IT Lab CCNA series.
Built a basic enterprise topology with two branches.

### Devices used
- Cisco 2911 Routers x2
- Cisco 2960-24TT Switches x2
- Cisco 5505 Firewalls x2
- PCs, Servers, and an attacker Laptop

### Topology
- New York Branch connected to Tokyo Branch via internet router
- Firewalls separating each branch from the internet
- Attacker laptop simulated on the internet segment

### What I learned
- Navigating Packet Tracer interface
- Placing and connecting network devices
- Using auto connection type feature
- Understanding basic enterprise network segmentation
