
This lab challenges you to design and implement a Variable Length Subnet Mask (VLSM) scheme using the 192.168.5.0/24 space to efficiently satisfy specific host requirements across four distinct LANs and a point-to-point router link.




You will apply end-to-end connectivity by configuring precise first/last usable IP addresses on host PCs and router interfaces, alongside implementing static routes to ensure complete inter-network communication.
