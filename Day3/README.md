
This lab uses Cisco Packet Tracer to demonstrate and analyze network traffic across multiple layers of the OSI model. The topology consists of two routers (R1, R2), two switches (SW1, SW2), a server (SRV1), and a client PC (PC1) spanning two networks — 192.168.1.0/24 and 10.0.0.0/24. Using Packet Tracer's Simulation mode, you can observe and analyze the various protocol types traversing the network in real time, including OSPF, STP, DHCP, ARP, and ICMP, giving you a practical understanding of how different protocols operate at their respective OSI layers.




By stepping through the simulation, you'll see how Layer 2 protocols like STP and ARP handle local network communication, how Layer 3 protocols like OSPF and ICMP manage routing and reachability between networks, and how higher-layer services like DHCP enable automatic address assignment. The lab also includes a Layer 7 traffic generation task to round out the full-stack view. It's designed as a hands-on companion to CCNA study material, making abstract OSI concepts tangible through live packet capture and event inspection.
